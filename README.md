# README

## Ruby version
ruby '2.5.8'

## Database creation
- createdb postgres
- rake db:create

## Database initialization
- rake db:migrate
- rake db:seed

## How to run the test suite
rspec spec

##  Startup
- rails s
- Navigate to localhost:3000
